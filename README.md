# E-Judiciary Case Management System Using Binary Search Tree (BST)
This project is a prototype e-Judiciary system designed to demonstrate how Binary Search Trees (BSTs) can improve the filing, indexing, searching, and management of case records. The prototype is written in Python and showcases how BST operations can support a more efficient judicial workflow.
